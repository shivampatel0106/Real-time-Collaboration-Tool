# Real-time Collaboration Tool
Technologies: Node.js, Socket.IO, Express.js, JavaScript, HTML, CSS
•	Developed a real-time collaboration tool by setting up a Node.js server with Express.js and Socket.IO for live communication.
•	Implemented bidirectional communication using Express.js and Socket.IO to handle interactions between server and clients.
•	Designed the user interface using HTML, CSS, and JavaScript, integrating Socket.IO for continuous server connectivity.
•	Managed user interactions such as sending document files and document editing, effectively emitting events to the server for seamless collaboration.
